# GALAXY Discord Bot Manager 🤖

A powerful, production-grade desktop dashboard designed to manage, monitor, and restart your Discord bots locally with a clean, modern UI.

![Galaxy Manager Interface](https://github.com/m205a/Galaxy-Discord-Bot-Manager/blob/main/icon.png)

## 🚀 Features

- **Local Bot Management**: Start, stop, and manage multiple bots from a single dashboard.
- **Production-Grade Monitoring**: Real-time tracking of RAM usage and Uptime for every bot.
- **Auto-Crash Recovery**: Automatically restarts bots if they crash, ensuring 24/7 uptime.
- **Discord.py Helper**: One-click library installation and updates.
- **Multilingual Support**: Switch seamlessly between Arabic and English.
- **Background Operation**: Keep your bots running in the System Tray without cluttering your taskbar.
- **Clean UI**: Built with `CustomTkinter` for a modern, dark-mode experience.

## 🛠️ Tech Stack

- **Python**: Core logic and automation.
- **CustomTkinter**: Modern GUI framework.
- **psutil**: System monitoring.
- **PyInstaller**: For single-file executable builds.

## 📥 Installation

### Option 1: Run from Source
1. Clone the repository: `git clone https://github.com/m205a/Galaxy-Discord-Bot-Manager.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python "Bot Manager.pyw"`

### Option 2: Use Executable
Download the latest `.exe` from the [Releases](https://github.com/m205a/Galaxy-Discord-Bot-Manager/releases) section.

## 📖 How to Use
1. Launch the application.
2. Click **"➕ Add New Bot"** and select your `bot.py` file.
3. Click **"Start"** to run your bot.
4. Use the **"Logs"** button to monitor console output in real-time.
