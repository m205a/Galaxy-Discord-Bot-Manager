# GALAXY Discord Bot Manager
A powerful, production-grade desktop dashboard to manage, monitor, and restart your Discord bots locally with a clean UI.

## Features
- 🚀 One-click start/stop for multiple bots.
- ⚡ Auto-install discord.py library.
- 📊 Real-time RAM and Uptime monitoring.
- 🔄 Auto-restart functionality on crashes.
- 🌍 Multilingual support (Arabic & English).
- 🤫 Background mode (System Tray).

## How to use
1. Download the latest release.
2. Run the application.
3. Add your bot's path via the "Add New Bot" button.
